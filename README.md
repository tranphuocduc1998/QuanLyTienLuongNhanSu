# Môn Quản lý Tiền Lương Và Nhân Sự
*** Phần Mềm ***

## Công cụ hỗ trợ
1. Visual studio
2. Dev express
2. Sql Server

## Chức Năng Phần Mềm
1. Đăng nhập: phần chia chức vụ đăng nhập `Admin`, `Quản lý nhân sự` và `Quản lý tiền lương` 
2. Chức năng Xem, Thêm, Xóa, Sửa cho từng tác vụ đăng nhập
    I. Nhân viên
    II. Phòng ban
    III. Chức vụ
    IV. Chấm công
    V. Lương cá nhân
    VI. Bồi dưỡng
    VII. Tạm ứng
    VIII. Khen thưởng và kỹ luật
    IX. Bảo hiểm
