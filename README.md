# Môn Quản lý Tiền Lương Và Nhân Sự
*** Phần Mềm ***

## Công cụ hỗ trợ
1. Visual studio
2. Dev express
2. Sql Server

## Chức Năng Phần Mềm
1. Đăng nhập: phần chia chức vụ đăng nhập `Admin`, `Quản lý nhân sự` và `Quản lý tiền lương` 
2. Chức năng Xem, Thêm, Xóa, Sửa cho từng tác vụ đăng nhập
    1. Nhân viên
    2. Phòng ban
    3. Chức vụ
    4. Chấm công
    5. Lương cá nhân
    6. Bồi dưỡng
    7. Tạm ứng
    8. Khen thưởng và kỹ luật
    9. Bảo hiểm
